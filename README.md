<h1 align="center">Talent Show Submission App </h1> <p align="center"> A PyQt desktop application that allows users to submit their videos for a talent show, while providing administrators with tools to view and manage participant submissions. </p>

# Features

# User Side

Submit talent show videos through a simple desktop interface

Upload participant information (name, category, description, etc.)

Secure and user-friendly submission process

# Admin Side

Access the list of all participants

View and download submitted video content

Manage and review submissions from a centralized dashboard

# Tech Stack

Python

PyQt5

SQLite / MySQL (or any supported database) / MongoDb for videos 

File system storage for videos

# Application Workflow

Users open the application and submit their video entries

Videos and participant metadata are stored securely

Admins log in to access the list of participants

Admins can view, review, and download submitted videos

# Installation

Clone the repository:

git clone https://github.com/yourusername/talent-show-app.git
cd talent-show-app


# Install dependencies:

pip install pyqt5


# Run the application:

python talent_show.py
python admin.py
